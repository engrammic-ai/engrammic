## Bugs Found

### BUG-001: Graph traversal returns unrelated nodes (MEDIUM)

**Scenario:** 006-wisdom-layer  
**Symptom:** `context_recall` with `node_ids` and `depth=2` returns `edges_traversed: 0` and 50 unrelated Commitment nodes instead of following DERIVED_FROM edges from the seed node.

**Root cause (suspected):** Cypher query in `context_service/services/context.py:1595-1613` uses property-only MATCH without label, failing to find nodes that have a specific label (e.g., `Document`).

**Impact:** Graph traversal from any seed node does not work.

### BUG-002: belief_history superseded_by field inverted (LOW)

**Scenario:** 009-temporal-queries  
**Symptom:** When querying belief_history from the NEW (active) node, its `superseded_by` field contains the OLD node's ID instead of null. Status also shows "superseded" on the active node.

**Impact:** Callers may misinterpret which node is current in a supersession chain.

### BUG-003: create_silo action not available (LOW)

**Scenario:** 003-silo-isolation  
**Symptom:** `context_admin(action="create_silo")` returns invalid action. Only one silo exists and cannot create additional silos dynamically.

**Impact:** Cannot fully test multi-silo isolation without seeding silos externally.

### BUG-004: ReasoningChain nodes invisible to flat GET (MEDIUM)

**Scenario:** 012-reasoning-chains  
**Symptom:** `context_recall` with `node_ids=[<chain_id>]` returns `node_not_found` even though graph traversal confirms the node exists.

**Root cause:** `GET_NODES_BY_IDS_TEMPORAL` query cannot find `ReasoningChain` nodes. The `include_steps=true` feature never fires because the node lookup fails first.

**Impact:** Cannot retrieve reasoning chain steps via the documented API.

### BUG-005: close_session feature-flagged off (LOW)

**Scenario:** 012-reasoning-chains  
**Symptom:** `context_admin(action="close_session")` returns `feature_disabled` error.

**Impact:** Session compaction not available.
