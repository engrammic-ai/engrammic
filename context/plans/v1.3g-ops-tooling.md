## Tasks

### Tombstone Asset

1. Implement `causal_tombstone` Dagster asset
2. Accept edge ID list OR filter criteria:
   - `silo_id` (required)
   - `edge_type` (optional: CAUSES, CORROBORATES, PREVENTS)
   - `confidence_below` (optional: float threshold)
   - `created_before` (optional: datetime)
3. Tombstone edges and their derived inferences (use `inferred_from_edge_ids` reverse lookup)
4. Add manual trigger via admin API endpoint (`POST /admin/tombstone`)
5. Respect silo boundaries (no cross-silo tombstoning)

### Confidence Metrics

1. Add Prometheus histograms:
   - `edge_confidence_distribution` (buckets: 0.1, 0.3, 0.5, 0.7, 0.9)
   - `belief_confidence_distribution` (same buckets)
   - Labels: `silo_id`, `edge_type`
2. Emit on write operations
3. Add Dagster sensor for drift alerting:
   - Trigger if mean confidence drops > 0.1 from baseline
   - Log warning, don't block
