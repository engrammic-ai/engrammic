## Scope

| Component | Description |
|-----------|-------------|
| `context_close_reasoning` | MCP tool to explicitly close a reasoning session |
| Session state | Track session state (open, closed, summarized) on chain nodes |
| Cross-chain linking | Handle references between reasoning chains in `context_graph` |

## Design Decisions (locked)

| Question | Decision | Rationale |
|----------|----------|-----------|
| Session grouping | Single chain = single session | Multi-chain sessions deferred to v1.4 |
| Auto-close timeout | No auto-close in v1.3 | Explicit close only; auto-close is v1.4 |
| Cross-chain references | Store as REFERENCES edge | Consistent with existing edge model |

## Tasks

1. **Add `session_state` property** to reasoning chain nodes (open/closed/summarized)
2. **Implement `context_close_reasoning` MCP tool** - calls existing `compact_reasoning_chain`, sets state
3. **Extend `context_graph`** to traverse REFERENCES edges for cross-chain links
4. **Extend `context_history`** to include cross-chain references in provenance
5. **Trigger summarization on close** if chain > `compaction_step_threshold` (use existing summarization.py)
6. **Gate behind `session_compaction_enabled` flag**
7. **Tests**
