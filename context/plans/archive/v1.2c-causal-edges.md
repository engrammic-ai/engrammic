## Design Decisions (locked 2026-05-03)

See [v1.2c-brainstorm-synthesis.md](./v1.2c-brainstorm-synthesis.md) for full analysis.

| Decision | Choice |
|----------|--------|
| Inference location | Dagster asset (batch) |
| Storage model | Materialized with `inferred=true` |
| Confidence formula | Multiplicative + source-dedup |
| Depth limit | 3 hops (configurable) |
| PREVENTS | Add to primitives before Phase 1 |
| Confidence fields | Dual: `extraction_confidence`, `consensus_confidence` |
| Feature flags | Three: extraction, inference, query |
| Backfill | New only + `causal_coverage_from` epoch |

## Phase 1: Configuration + Schema

Add to `src/context_service/config/settings.py`:

```python
class CausalConfig(BaseModel):
    extraction_enabled: bool = False
    inference_enabled: bool = False
    query_enabled: bool = False
    max_transitivity_depth: int = Field(default=3, ge=2, le=4)
    min_inferred_confidence: float = Field(default=0.5)
    confidence_formula: str = "multiplicative"
    transitivity_batch_size: int = 500

# In Settings:
causal: CausalConfig = Field(default_factory=CausalConfig)
```
