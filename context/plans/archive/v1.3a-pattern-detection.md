## Scope

| Pattern Type | Description | Detection Method |
|--------------|-------------|------------------|
| co_occurrence | Facts frequently appearing in same cluster | Leiden clustering analysis |
| causal_chain | A CAUSES B CAUSES C chains | Graph traversal on CAUSES edges |
| pattern_decay | Reduce pattern confidence if not re-observed | Time-based decay on pattern nodes |
