## New Flags Required

| Flag | Config Location | Phase | Purpose |
|------|-----------------|-------|---------|
| `pattern_detection_enabled` | PatternConfig (new) | 3a | Gate co_occurrence + causal_chain detection |
| `llm_pattern_enabled` | PatternConfig | 3b | Gate LLM pattern detection (separate from rule-based) |
| `session_compaction_enabled` | CompactionConfig (extend) | 3c | Gate `context_close_reasoning` |
| `reflection_triggers_enabled` | AutoReflectConfig (extend) | 3d | Gate confidence/contradiction/uncertainty triggers |

Note: CausalConfig flags already exist for 3e work.
