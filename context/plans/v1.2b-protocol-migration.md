## Migration Pattern

Each file should:
1. Import from `engine.protocols` instead of `stores.memgraph` / `engine.memgraph_store`
2. Accept `HyperGraphStore` protocol in function signatures
3. Use protocol methods only (no Memgraph-specific calls)

Note: `execute_query`/`execute_write` are escape hatches on the protocol. Using them is allowed but counts as tech debt (still coupled to Cypher dialect).
