   - Rewrote to OPTIONAL MATCH + IS NULL pattern

   - Rewrote to OPTIONAL MATCH approach

3. **size() on patterns**: `size((n)-[:REL]->())` not reliable
   - Replaced with OPTIONAL MATCH + count()

   - Fixed in synthesizer queries
