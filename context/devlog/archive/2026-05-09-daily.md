## MCP Client: Engrammic Rebrand

Rebranded `delta-prime-mcp` to `engrammic-mcp` for MCP registry publishing.

### Changes

- Package renamed: `delta-prime-mcp` -> `engrammic-mcp` v0.2.0
- Module renamed: `delta_prime_mcp` -> `engrammic_mcp`
- Env vars: `DELTA_PRIME_*` -> `ENGRAMMIC_*`
- Credentials path: `~/.delta-prime/` -> `~/.engrammic/`
- Backend URL default: `api.engrammic.com`

### New Tools (6)

- `context_belief_state` - query session WorkingHypotheses
- `context_update_belief` - mutate hypothesis confidence
- `context_crystallize` - promote hypotheses to commitments
- `context_accept_belief` - accept ProposedBelief
- `context_reject_belief` - reject ProposedBelief
- `context_skills` - skill registry access (list/get/search)

### CLI

- `engrammic-mcp login` - OAuth browser flow with local callback server
- `engrammic-mcp --version` / `--help`
- Startup health check via `context_admin(action="whoami")`
